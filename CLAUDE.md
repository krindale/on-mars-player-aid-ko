# On Mars Player Aid — 한글 번역 프로젝트 규칙

## 🎯 목표
PDF 문서를 **구조를 유지하면서** 자연스럽게 번역하는 자동화 파이프라인을 구축한다.

## 🧩 전체 구조
```
PDF 입력
  ↓
텍스트 + 레이아웃 추출 (PyMuPDF)
  ↓
문단/구조 분리 (제목·본문·표 분류)
  ↓
번역 (Claude 가 직접)
  ↓
결과 병합 (원본 좌표·스타일 유지)
  ↓
PDF 출력
```

## 1️⃣ PDF → 구조 추출
- **PyMuPDF** 로 텍스트 + bbox + 폰트 + 색상 추출
- 표는 `pdfplumber` 보조
- 단순 텍스트 추출 ❌ → **블록 단위 추출 ✅**
- 텍스트가 outline(vector path) 으로 변환된 PDF 인 경우 페이지를 이미지로 렌더링한 뒤 시각적으로 구역을 정의하고 좌표를 직접 매핑

## 2️⃣ 구조 재구성
```json
[
  {"type": "heading", "text": "..."},
  {"type": "paragraph", "text": "..."}
]
```
문단 단위 분리, 제목/본문 구분, 표 별도 처리.

## 3️⃣ 번역 파이프라인
- 문단 단위 번역, 일관된 프롬프트
- 용어집 적용으로 일관성 유지, 긴 문단은 분할
- 번역 프롬프트 원칙
  ```
  다음 텍스트를 한국어로 번역하라.
  조건:
  1. 문단 구조 유지
  2. 줄바꿈 유지
  3. 제목/본문 구분
  4. 자연스러운 번역
  5. 요약 금지
  ```

## 4️⃣ 표 처리
표를 일반 텍스트로 번역하면 깨짐 → 셀 단위 분해 → 각 셀 번역 → 구조 재조립.

## 5️⃣ 결과 병합
- 원본 PDF 의 시각 요소(배경 이미지, 아이콘, 띠 등)는 보존
- 영문 텍스트만 redaction(흰색 마스킹) 으로 제거 후 같은 좌표에 한글 삽입
- 줄 간격·정렬·색상 유지

## ⚠️ 실패 패턴
- ❌ 전체 문서 한 번에 번역 / 구조 무시 / 표 미처리
- ✅ 블록 단위 처리 / 구조 기반 번역 / 단계별 파이프라인

## 🚀 확장
OCR(스캔 PDF), 다국어, 품질 평가 모듈, 웹 UI.

## 🧠 핵심 요약
> **PDF 번역은 번역 문제가 아니라 구조 문제다.**
> 구조 추출 70% / 번역 20% / 후처리 10%

## 📌 추천 개발 순서
1. 텍스트 추출 → 2. 문단 분리 → 3. 번역 → 4. PDF 출력 → 5. 표 처리 개선

# 본 프로젝트 작업 규칙

- **번역 주체**: Claude 가 **직접** 번역. 외부 LLM API 호출 금지.
- **폰트**: 모든 한글 본문/제목은 `fonts/SUIT/` 의 SUIT 패밀리 사용
  - 제목/강조 → `SUIT-Bold.otf` 또는 `SUIT-ExtraBold.otf`
  - 본문 → `SUIT-Regular.otf` / `SUIT-Medium.otf`
- **레이아웃**: 원본 페이지 사이즈를 그대로 유지 (이 문서의 경우 1008×612pt 가로형)

## 디렉토리

```
data/
  paragraphs_ko.json   # 번역 데이터 + 좌표
  font_map.json        # 섹션별 폰트/사이즈/색상 매핑
src/
  render.py            # 메인 렌더러
  pages/               # 페이지별 구역 정의
fonts/SUIT/            # 한글 폰트
out/
  pages_en/            # 원본 페이지 PNG
  verify/              # 시각 검증 결과
  OnMars-PlayerReference-v07_KO.pdf
```

## 작업 절차
1. PDF 를 페이지별로 PNG 렌더링 (DPI 200+)
2. 시각 분석으로 모든 텍스트 구역 좌표 추출 → `src/pages/page_NN.json`
3. 영문 원문 받아쓰기 → `data/paragraphs_en.json`
4. 한국어 번역 → `data/paragraphs_ko.json`
5. 원본 페이지를 배경으로, 영문 영역을 흰색으로 마스킹, 한글 오버레이 → 새 PDF
6. `out/verify/` 에서 시각 비교 검증
